
1)
C++ — компилируемый, мультипарадигмальный, статически типизированный язык программирования общего назначения,
на котором можно создавать программы любого уровня сложности.
> мультипарадигмальный - включающий широкий спектр различных стилей и технологий программирования
> компилируемый -  написанный код преобразуется компилятором в машинный код и записывается в исполняемый файл
> статически типизированный: все переменные и функции имеют свой неизменный тип

2)
Фундаментальный типы данных:
> void - отсутствие типа
> double - 8 байт, цисло с плавающей точкой
> bool - 1 байт, значения true/
> char - 1 байт, символьный тип
> float - 4 байта, цисло с плавающей точкой
> int - 4 байта, целочисленный тип

3)
Переносимость — это свойство, которое указывает, насколько легко можно перенести код,
написанный на одной платформе, на другую. Проблема переносимости кода возникает при его переносе
между различными программно-аппаратными платформами. Изначальное создание
переносимого продукта позволяет достаточно безболезненно портировать его на новые платформы.

4)
> Определение - x = 0; присваивание некторого значения
> Объявление - int x; "команда" процессору выделить часть памяти на переменную
> Инициализация - int x = 0; объявление + определение

5)
Выравнивание ускоряет доступ к памяти за счет генерации кода, в котором на чтение и запись ячейки
памяти требуется по одной инструкции. Без выравнивания мы можем столкнуться с ситуацией, когда процессору
придется использовать две и более инструкции для доступа к данным, расположенным между адресами, кратными
размеру машинного слова.
